Système hors ligne basé sur SQL Server

Contrairement à de nombreuses solutions cloud, ce logiciel fonctionne localement avec SQL Server et des sauvegardes SQLite.

  • Aucun abonnement obligatoire
  • Aucune dépendance aux pannes du cloud
  • Propriété complète des données
  • Génération de sauvegardes SQLite sécurisées

Vos données d’entreprise restent sous votre contrôle.

Questions fréquentes

Le logiciel est-il gratuit ?
Oui. Le système de bureau principal est entièrement gratuit à télécharger et à utiliser.

Non. Le logiciel de bureau fonctionne hors ligne. Internet est uniquement nécessaire pour la synchronisation mobile.

Puis-je sauvegarder mes données ?
Oui. Vous pouvez générer des sauvegardes SQLite pour la sécurité.

Convient-il aux entreprises en croissance ?
Oui. Sa structure permet de gérer l’augmentation des transactions et les opérations mobiles.
